# 🐼 Flappy Panda

Un jeu de type Flappy Bird où l'oiseau est remplacé par un panda mignon. Esquivez les bambous le plus longtemps possible !

## Jouer

[▶ Jouer en ligne](https://lunatii82.github.io/Weekly-game/floppy-panda/)

## Contrôles

| Action | Commande |
|--------|----------|
| Sauter | `Espace` |
| Sauter | Clic souris |
| Sauter | Tap (mobile) |

## Fonctionnalités

- Panda dessiné entièrement en Canvas
- Obstacles sous forme de bambous
- Le meilleur score est sauvegardé localement dans votre navigateur (localStorage)
- Fonctionne sans installation, sans dépendance externe

## Lancer localement

Double-cliquez simplement sur `index.html` — aucun serveur requis.
